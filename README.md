# 🛒 ShopFlow

A modern and responsive mini e-commerce web project built using HTML and CSS.

## 🚀 Features

* Responsive navigation menu
* Animated background UI
* Sign In & Register صفحات
* Product page with:

  * Image display
  * Size selection
  * Quantity selection
  * Price display
  * Add to Cart button
* Payment form:

  * Card Number
  * Card Holder Name
  * Expiration Date
  * CVV validation (numbers only)

## 🧱 Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)

## 🎯 Project Purpose

This project was created as a practice to improve front-end development skills and build a clean and modern user interface.

## 📌 Future Improvements

  * Form validation

  * Dynamic price calculation
