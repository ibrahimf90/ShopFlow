## ShopFlow

A modern and responsive mini e-commerce web project.

## Features

* Responsive navigation menu
* Animated background UI
* Pages Sign In & Register 
* Product page with:
  * Image display
  * Size selection
  * Quantity selection
  * Price display
  * Add to Cart button
* Payment form:
  * Card Number
  * Card Holder Name
  * Expiration Date
  * CVV validation (numbers only)

## Technologies Used

* HTML5
* CSS3 (Flexbox & Grid)
